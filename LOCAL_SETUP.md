# Как запустить сайт локально

Краткий гайд по запуску проекта Prism на своём компьютере.

---

## 1. Что нужно заранее

- **Node.js** (рекомендуется версия 18 или 20)  
  Проверить: откройте терминал и введите:
  ```bash
  node -v
  ```
  Если команда не найдена — установите Node.js с [nodejs.org](https://nodejs.org/).

- **npm** (обычно ставится вместе с Node.js)  
  Проверить:
  ```bash
  npm -v
  ```

---

## 2. Открыть папку проекта

Откройте терминал (PowerShell, cmd или «Терминал» в VS Code/Cursor) и перейдите в папку проекта:

```bash
cd C:\Users\Айрат\Desktop\portik
```

Если проект лежит в другом месте — укажите свой путь вместо этого.

---

## 3. Установить зависимости

Один раз выполните:

```bash
npm install
```

Дождитесь окончания установки. Появится папка `node_modules` и файл `package-lock.json`.

---

## 4. Запустить сайт в режиме разработки

Выполните:

```bash
npm run dev
```

В терминале появится что-то вроде:

```
  VITE v6.x.x  ready in xxx ms
  ➜  Local:   http://localhost:3000/
  ➜  press h + enter to show help
```

---

## 5. Открыть сайт в браузере

Откройте в браузере адрес:

**http://localhost:3000**

Сайт будет обновляться при сохранении файлов (hot reload).

Остановить сервер: в терминале нажмите **Ctrl + C**.

---

## Дополнительно

| Действие | Команда |
|----------|--------|
| Собрать проект для продакшена | `npm run build` |
| Посмотреть собранную версию | `npm run build` затем `npm run preview` |

После `npm run build` результат будет в папке `dist/`.  
После `npm run preview` сайт откроется по адресу вроде http://localhost:4173.

---

## Если что-то пошло не так

- **«npm не является внутренней или внешней командой»** — Node.js не установлен или не добавлен в PATH. Переустановите Node.js с официального сайта.
- **Ошибка при `npm install`** — проверьте интернет и повторите команду. Если ошибка про конкретный пакет — пришлите текст ошибки.
- **Порт 3000 занят** — в проекте в `vite.config.ts` задан `port: 3000`. Можно изменить порт в этом файле или закрыть программу, которая использует 3000.
